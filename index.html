Forkify App

I built this project as a part of the Complete Javascript course 2024 from Udemy. Users are able to search for recipes and are able to add their own. Recipes are from the Forkify API.

I really enjoyed that we were working with an API. It was pretty cool to see the amount of recipes that would be returned from one search. And because of Javascript, all of the information was formatted in a user-friendly way on the page.
